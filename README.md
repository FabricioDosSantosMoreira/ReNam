# ReNam
ReNam is a renaming tool for movies, series and animes using TMDB

- NOTE: ReNam is a basic tool, it may or may not work. I intend to update to fix and add new things. Thanks :)
 

# HOW TO USE

- Go to https://www.themoviedb.org/ and create an account
- Now go to https://www.themoviedb.org/settings/api and create an API key
- Go to ApiHandler.py and insert in API_KEY your new api key
- Run main.py
- NOTE: before renaming anything, select the directory path
- NOTE: feel free to add new file_format and patterns in MidiaHandler
